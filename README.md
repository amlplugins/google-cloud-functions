# @amlplugins/google-cloud-functions

Native `@google-cloud/functions` SDK re-exported under the `@amlplugins` namespace.

> Cloud Functions.

## Install

```bash
npm install @amlplugins/google-cloud-functions
```

## Use

```ts
import { /* named exports from @google-cloud/functions */ } from "@amlplugins/google-cloud-functions";
// or
import sdk from "@amlplugins/google-cloud-functions";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
